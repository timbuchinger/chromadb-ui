# Dark/Light Mode

As a user, I want to be able to trigger between light and dark for the colors used in the app.

## Technical Requirements

1. This should be implemented with a toggle in the navbar.

## Acceptance Critera

1. The user can toggle between light mode and dark mode.
2. The default selection is based on the users' system preference.
3. Their theme selection is saved for the next time they visit the website.
